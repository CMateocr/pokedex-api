# DEV ENVIRONMENT
1. Clone repository
2. Execute
```
yarn install
```
3. Nest CLI already installed
```
npm i -g @nestjs/cli
```
4. Run DB
```
docker compose up -d
```

## Stack used
* MongoDB
* Nest
